The this keyword in functions behaves differently in strict mode. The this keyword refers to the object that called the function. Strict mode does not allow undeclared variables.
Also 
    Future Proof!
Keywords reserved for future JavaScript versions can NOT be used as variable names in strict mode.

These are:

implements
interface
let
package
private
protected
public
static
yield

With "use strict":

Using a variable without declaring it, is not allowed.

Activate debugging in your browser (F12) to see the error report.
